# Scripts

סקריפטי הרצה לאפליקציית WhatsApp Bot.

## Linux/Mac

- `run.sh` - הרצה במצב test
- `run-production.sh` - הרצה במצב production

שימוש:
```bash
./scripts/run.sh
# או
./scripts/run-production.sh
```

## Windows

### PowerShell
- `run-test.ps1` - הרצה במצב test
- `run-production.ps1` - הרצה במצב production

שימוש:
```powershell
.\scripts\run-test.ps1
# או
.\scripts\run-production.ps1
```

### Command Prompt (CMD)
- `run-test.bat` - הרצה במצב test
- `run-production.bat` - הרצה במצב production

שימוש:
```cmd
scripts\run-test.bat
# או
scripts\run-production.bat
```
